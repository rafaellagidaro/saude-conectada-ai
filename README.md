## 🚀 Visão Geral
Este não é apenas um portal de saúde. É um ecossistema híbrido que combina a performance do **C++** para validação de sensores, a flexibilidade do **Node.js** para gestão de APIs e **Machine Learning Preditivo** para antecipação de crises clínicas.

### 🧠 Diferenciais Tecnológicos (The "Pro" Stack)
* **Native Bridge (C++):** Validação de sinais vitais em baixo nível para garantir integridade biológica.
* **Predictive AI:** Algoritmo de Regressão Linear para análise de tendências de glicémia e pressão.
* **Audit Ledger (SHA-256):** Cada registo médico é assinado digitalmente para garantir imutabilidade.
* **PWA Ready:** Instalável em dispositivos móveis com suporte offline via Service Workers.

## 🛠️ Estrutura do Projeto
- `/` : Frontend SPA (Single Page Application) e ficheiros PWA.
- `/backend` : API REST, Middlewares de segurança e Motor de ML.
- `/core-native` : Código-fonte C++ para validação de alta performance.
- `/database` : Esquemas SQL com suporte a auditoria digital.
